# Evidence

所有投研 skills 统一使用以下**完整中文证据标签**，避免把猜测写成事实。

- `公开事实`：公开、稳定、可直接验证的事实
- `财报披露`：财报、公告、官方披露、权威数据库中的明确数字或事实
- `市场共识`：市场观点、卖方一致预期或普遍流行的预期口径
- `合理推演`：基于已知事实做出的推导、换算或判断，必须说明链路
- `待核验假设`：传闻、线索、渠道信息或仍需进一步核验的假设

## 使用原则

- 关键结论优先由 `公开事实`、`财报披露` 支撑
- 可以使用 `市场共识` 描述预期，但不能把共识当事实
- `合理推演` 必须说明推演链路
- `待核验假设` 不能成为最终结论的唯一依据

## 标注要求

- 证据标签**一律用完整中文**，不使用任何字母缩写，也不要字母 + 中文混排
- 每条关键事实、每个具体数字都要带标签
